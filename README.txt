Este software é um simulador didático de manipuladores robóticos.

O programa foi desenvolvido pelo Departamento de Engenharia Elétrica da Universidade Federal de Viçosa.

O programa foi desenvolvido para fins acadêmicos. Os autores não se responsabilizam por nenhum dano que o software possa causar e não têm a obrigação de prover nenhum suporte ou documentação adicional para sua utilização.

O desenvolvimento foi feito em C++ usando bibliotecas gráficas do OpenGL no ambiente de desenvolvimento do Borland C++ Builder 6.0.

A pasta principal do projeto é a ManipXP_serial.

O arquivo de projeto principal é Manipulador.bpr e pode ser aberto através da IDE do Borland C++ Builder.

O arquivo Manipulador.exe é uma versão compilada do programa para Windows.

O arquivo Ajuda.chm possui uma documentação básica para utilizar o software.
